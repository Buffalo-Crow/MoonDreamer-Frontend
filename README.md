# MOONDREAMER - Dream Diary - Astrology - AI - COMMUNITY

## ABOUT 
- Moondreamer is a full stack application that incorporates a few different worlds into one seemless user experience.
- MD is a Dream Diary that calculates the moon sign in which the users dream occured in. Over time the user can build data and begin to see patterns. 
- The user can keep their dreams private or share them to a social media like feed where other users can comment or like the dream that was shared. 
- AI provides insights to the users dream with a click of a button. You can save the insight and genreate a new insight. 
- This project incorporates AI, social media in a really new way that has never been done before.
